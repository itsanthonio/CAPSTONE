Legal Concessions — GeoJSON Upload

Boundaries of licensed mining concessions from the Minerals Commission. Uploading replaces all existing concessions and automatically reclassifies every detection.

Drop GeoJSON file here or click to browse

Accepted: .geojson, .json  ·  Polygon or MultiPolygon features

What happens on upload

  1. 1 All existing concession boundaries are replaced with the new file
  2. 2 Every detection is checked against the new boundaries using a spatial intersection
  3. 3 Detections inside a concession → Legal. Outside → Illegal
  4. 4 The live map updates immediately to reflect the new classification

Expected GeoJSON properties: license_number, name, holder_name, license_type. Missing fields use sensible defaults.
